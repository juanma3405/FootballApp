-Aplicación realizada con C#. Modelo Mvc. 
-Consume api de futbol https://api.football-data.org/v4/ para obtener los resultados de la fecha del torneo actual que se indique de las principales ligas europeas 
(Alemania, España, Francia, Primera y Segunda de Inlaterra, Italia, Paises Bajos, Portugal) y Brasil.

--Tecnologias--
-Net 6.0
-Microsoft.AspNetCore.Identity.EntityFrameworkCore 6.0.16
-Microsoft.EntityFrameworkCore.SqlServer 7.0.5
-Microsoft.EntityFrameworkCore.Tools 7.0.5
